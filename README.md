<h1>Automatic Release Note Generator</h1>

<p>
A Windows Forms based tool created to generate clean and stylish release notes automatically.  
It removes the need for manual formatting and reduces the entire process from thirty to forty minutes to just one second.
</p>

<h2>Key Features</h2>
<ul>
    <li>Generates fully formatted release notes automatically.</li>
    <li>Uses the ClosedXML library to write an Excel file with proper styles and formatting.</li>
    <li>Simple Windows Forms interface with fast input and export.</li>
    <li>Supports multiple modules and dynamic release note creation.</li>
</ul>

<h2>Technologies Used</h2>
<ul>
    <li>C sharp</li>
    <li>Windows Forms</li>
    <li>WPF</li>
    <li>ClosedXML</li>
</ul>
